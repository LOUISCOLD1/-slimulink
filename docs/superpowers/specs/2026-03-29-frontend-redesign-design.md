# 牧智通前端重设计 - 设计规格

## 项目定位

重新设计微信小程序前端，保持现有后端 API 不变，提升 UI/UX 质量，使其更适合目标用户（40-70 岁蒙古族牧民）并在比赛中具有视觉辨识度。

## 目标用户画像

- 内蒙古农牧民，40-70 岁
- 文化程度偏低，可能不识汉字
- 蒙古语为母语，汉语为第二语言
- 会用微信，但不太会打字
- 视力可能不好，需要大字体

## 设计决策

### 1. 视觉风格：草原绿金（B+ 方案）

- **主色调：** 深绿 `#064e3b` ~ `#047857`（导航栏、banner）
- **强调色：** 金色 `#fbbf24`（标题、边框、分割线）
- **背景色：** 浅绿 `#f0fdf4`（页面背景）
- **卡片底色：** 白色 `#ffffff`
- **装饰元素：** banner 底部波浪分割线
- **按钮风格：** 圆角绿色，录音按钮带金色边框
- **提醒条：** 金色背景 `#fffbeb` + `#fcd34d` 边框
- **蒙汉双语：** 所有用户可见文案均蒙汉对照

### 3. 录音交互：点击开始/点击结束

- 点击一下麦克风按钮开始录音，按钮变红色+脉冲动画
- 再点击一下停止录音，自动上传识别
- 比"按住说话"对手指不灵活的老年人更友好

### 5. 便民电话：分类列表 + 收藏置顶

- 按类别分组：村级 → 乡镇 → 旗县 → 热线
- 每个号码卡片：名称（蒙汉）+ 描述 + 大号拨打按钮
- 收藏功能：常用号码可以收藏，置顶显示在"⭐ 常用号码"分组
- 收藏状态存本地 Storage

### 6. 设置页

包含以下选项：
- **🌐 语言切换** — 中文 / ᠮᠣᠩᠭᠣᠯ ᠬᠡᠯᠡ
- **🤖 AI引擎** — 智谱（免费）/ DeepSeek
- **🔤 字体大小** — 大 / 特大 / 超大（三档可选）
- **🔊 语音语速** — 慢速 / 正常 / 快速
- **🗑️ 清除对话历史** — 确认后清除所有对话记录
- **📱 关于牧智通** — 版本号 + 简介（蒙汉双语）

### 7. 数据存储变更

新增本地 Storage 字段：
- `chatHistory` — 对话历史列表 `[{id, title, messages[], createdAt, updatedAt}]`
- `favoriteContacts` — 收藏的电话号码 ID 列表
- `fontSize` — 字体大小档位 `"large" | "xlarge" | "xxlarge"`
- `speechRate` — 语音语速 `"-20%" | "+0%" | "+20%"`

### 8. 后端 API 变更

当前后端 API 无需修改，所有变更均在前端完成。对话历史存储在本地 Storage 中，不需要后端支持。

## 不包含在本次设计中

- 微信模板消息推送（后续迭代）
- 蒙语语音识别 STT（依赖讯飞接入）
- 自定义 tabBar（原生 tabBar 暂时够用）
- 用户登录/注册系统

## 技术约束

- 微信小程序原生开发（WXML + WXSS + JS）
- 不引入额外框架（如 Taro/uni-app）
- 后端 API 保持不变
- 兼容微信基础库 2.25.0+
